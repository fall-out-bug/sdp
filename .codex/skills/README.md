# Project-level skills (Codex)

SDP project skills are defined in `.claude/skills/` and invoked via SDP CLI (`sdp`) and Cursor/Claude skills.

- **@build** — Execute workstream (TDD, guard). See `.claude/skills/build/SKILL.md`.
- **@design** — Plan workstreams. See `.claude/skills/design/SKILL.md`.
- **@review** — Multi-agent quality review. See `.claude/skills/review/SKILL.md`.

Full list: `.claude/skills/` (build, design, feature, guard, oneshot, review, tdd, etc.).
