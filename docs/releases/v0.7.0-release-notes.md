## v0.4.0 — PRD Command + Two-Stage Review (2026-01-27)

### F011: PRD Command

- PRD command с project type profiles (cli, service, library)
- Авто-диаграммы из code annotations
- `@prd:` annotation parser
- Line limits validator, CodeReview hook integration

### F003: Two-Stage Review

- Stage 1: Spec Compliance (goal, AC coverage, alignment)
- Stage 2: Code Quality (coverage ≥80%, mypy, AI-readiness)
- Stage 2 только если Stage 1 прошёл

### F004: Platform Adapters

- PlatformAdapter interface
- Claude Code, Cursor, OpenCode adapters
- detect_platform() — auto IDE detection

### F005: Extension System

- sdp.local/, ~/.sdp/extensions/{name}/
- extension.yaml manifest
- ExtensionLoader — auto-discovery

### F006: Core SDP

- Workstream parser (PP-FFF-WW)
- Feature decomposition, Project map
- Pip package, file size reduction, integration tests

### F007: Oneshot & Hooks

- /oneshot — autonomous feature execution
- Git hooks (pre-commit, post-commit, pre-push)
- /debug, /test, /idea, /design skills

### F008: Contract-Driven WS Tiers

- Starter, Standard, Advanced tiers
- Capability tier validator, Model mapping registry
- Tier auto-promotion, Escalation metrics

### F010: SDP Infrastructure

- Submodule support
- PP-FFF-SS naming convention
- Content synchronization
