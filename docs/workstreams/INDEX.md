## Workstream ID Format

`PP-FFF-SS` — Project (00), Feature (001–052), Step (01, 02, …)

Example: `00-028-01` = sdp_lab, F028 CI Cleanup, step 1
Example: `00-014-01` = sdp_lab, F014 CI Loop CLI, step 1 (poll + classify)
