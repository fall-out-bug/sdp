# WS: Hook Packaging & Discovery

**Feature**: F077 - Runtime Hooks Platform
**ID**: 00-077-05
**Priority**: P1
**Depends on**: 00-077-01

## Goal
Add hook packaging model with metadata and discovery.

## Scope Files
- `src/sdp/hooks/package.go` (NEW)
- `src/sdp/hooks/package_test.go` (NEW)
- `src/sdp/hooks/discovery.go` (NEW)

## Acceptance Criteria
1. Hook pack metadata format (name, version, events, priority)
2. Hook discovery from `.sdp/hooks/` directory
3. Hook eligibility checks (version, dependencies)
4. Hook installation/uninstallation
5. Hook list command in CLI
6. Tests for packaging (>= 80% coverage)

## Out of Scope
- Specific hook types (00-077-02..04)

## Implementation Notes
