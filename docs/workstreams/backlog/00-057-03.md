# 00-057-03: sdp log show & Filters

## Goal

Extend `sdp log` (from F054) with interactive browser, filters, and export.

## Context

F054 WS-07 builds `sdp log trace` and `sdp log` (recent 20). This WS adds `sdp log show` (interactive browser), filtering by type/model/date, and CSV/JSON export.

## Acceptance Criteria

- [ ] AC1: `sdp log show` displays paginated event list (20 per page)
- [ ] AC2: `sdp log show --type=generation` filters by event type
- [ ] AC3: `sdp log show --model=claude-sonnet-4` filters by model
- [ ] AC4: `sdp log show --since=2026-02-01` filters by date
- [ ] AC5: `sdp log show --ws=00-054-03` filters by workstream
- [ ] AC6: `sdp log export --format=csv` exports for analysis
- [ ] AC7: `sdp log export --format=json` exports as JSON array
- [ ] AC8: `sdp log stats` shows summary (event counts by type, model distribution)
