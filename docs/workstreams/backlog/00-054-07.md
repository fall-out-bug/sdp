# WS: Acceptance & Approval Events

**Feature**: F054 - Evidence Layer
**ID**: 00-054-07
**Priority**: P0
**Depends on**: 00-054-01

## Goal
Implement acceptance and approval event types.

## Scope Files
- `src/sdp/evidence/acceptance.go` (NEW)
- `src/sdp/evidence/acceptance_test.go` (NEW)
- `src/sdp/evidence/approval.go` (NEW)
- `src/sdp/evidence/approval_test.go` (NEW)

## Acceptance Criteria
1. AcceptanceEvent for smoke test results
2. ApprovalEvent for human/auto approval
3. Approver field (who approved)
4. Reasoning/rationale capture
5. Tests for both events (>= 80% coverage)

## Out of Scope
- Other event types

## Implementation Notes
```go
type AcceptanceEvent struct {
    BaseEvent
    Command    string `json:"command"`
    Passed     bool   `json:"passed"`
    Output     string `json:"output"`
    Timeout    int    `json:"timeout_seconds"`
    Workstream string `json:"workstream,omitempty"`
}

type ApprovalEvent struct {
    BaseEvent
    Approver   string `json:"approver"`
    Mode       string `json:"mode"` // "auto" or "human"
    Reasoning  string `json:"reasoning,omitempty"`
    Workstream string `json:"workstream,omitempty"`
}
```

## Test Strategy
- Unit tests for both event types
- Test approver field
- Test mode handling
