## Implementation Notes

- The pattern: kubeopencode calls a webhook when Task transitions to terminal phase
- Webhook payload includes Task status, agent output, timing
- SDP's adapter-controller registers as a webhook receiver
- This makes SDP's evidence bridge reusable by anyone, not just us
