# WS: Onboarding Preflight Checks

**Feature**: F076 - Guided Onboarding Wizard
**ID**: 00-076-01
**Priority**: P0
**Depends on**: -

## Goal
Add preflight checks to `sdp init` command that detect project type and validate environment.

## Scope Files
- `sdp-plugin/cmd/sdp/init.go`
- `sdp-plugin/internal/init/preflight.go` (NEW)
- `sdp-plugin/internal/init/preflight_test.go` (NEW)

## Acceptance Criteria
1. Detects project type (go, node, python, mixed, unknown)
2. Checks for existing SDP configuration
3. Validates git repository status
4. Checks for conflicting files (.sdp, .claude)
5. Returns structured preflight result
6. Tests for all detectors (>= 80% coverage)

## Out of Scope
- Interactive prompts (00-076-02)
- Auto mode (00-076-03)
