# WS: Guided Fixes UX

**Feature**: F075 - Self-Healing Doctor
**ID**: 00-075-03
**Priority**: P0
**Depends on**: 00-075-01

## Goal
Add interactive guided fix mode with step-by-step instructions for manual issues.

## Scope Files
- `sdp-plugin/cmd/sdp/doctor.go`
- `sdp-plugin/internal/doctor/doctor_guided.go` (NEW)

## Acceptance Criteria
1. `sdp doctor --guided` prompts before each fix
2. Shows clear explanation of issue and proposed fix
3. Allows skip/fix/later options per issue
4. Generates fix script that can be saved and run later
5. Supports `--json` output for LLM consumption
6. Tests for guided mode (>= 80% coverage)

## Out of Scope
- Auto-repair (00-075-01)
- Deep checks (00-075-02)
