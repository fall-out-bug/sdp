# WS: Evidence Log Storage

**Feature**: F054 - Evidence Layer
**ID**: 00-054-02
**Priority**: P0
**Depends on**: 00-054-01

## Goal
Implement append-only evidence log storage with hash chain.

## Scope Files
- `src/sdp/evidence/log.go` (NEW)
- `src/sdp/evidence/log_test.go` (NEW)

## Acceptance Criteria
1. Append events to `.sdp/log/events.jsonl`
2. Hash chain linking (each event references previous hash)
3. Atomic writes (write-rename pattern)
4. File creation if not exists
5. Read all events from log
6. Tests for log storage (>= 80% coverage)

## Out of Scope
- Specific event types (00-054-03..08)
- CLI commands (00-054-09)
