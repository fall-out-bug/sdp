# WS: Evidence Export & Statistics

**Feature**: F054 - Evidence Layer
**ID**: 00-054-11
**Priority**: P0
**Depends on**: 00-054-08

## Goal
Add export and statistics functionality.

## Scope Files
- `src/sdp/evidence/export.go` (NEW)
- `src/sdp/evidence/export_test.go` (NEW)
- `src/sdp/evidence/stats.go` (NEW)
- `src/sdp/evidence/stats_test.go` (NEW)

## Acceptance Criteria
1. `sdp log export --format=json` - export to JSON
2. `sdp log export --format=csv` - export to CSV
3. `sdp log stats` - show event counts by type
4. `sdp log stats --by-date` - show events per day
5. Date range filtering
6. Tests for export/stats (>= 80% coverage)

## Out of Scope
- Remote storage
