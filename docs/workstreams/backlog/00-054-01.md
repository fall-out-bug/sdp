# WS: Evidence Event Types

**Feature**: F054 - Evidence Layer
**ID**: 00-054-01
**Priority**: P0
**Depends on**: -

## Goal
Define core evidence event types and base structures.

## Scope Files
- `src/sdp/evidence/event.go` (NEW)
- `src/sdp/evidence/event_test.go` (NEW)

## Acceptance Criteria
1. EventType enum with 6 types: decision, plan, generation, verification, acceptance, approval
2. BaseEvent struct with common fields (id, timestamp, type, prev_hash)
3. Event validation (required fields, valid types)
4. Hash calculation for event integrity
5. JSON serialization/deserialization
6. Tests for event types (>= 80% coverage)

## Out of Scope
- File storage (00-054-02)
- Event emission (00-054-03)

## Implementation Notes
```go
type EventType string

const (
    EventTypeDecision     EventType = "decision"
    EventTypePlan         EventType = "plan"
    EventTypeGeneration   EventType = "generation"
    EventTypeVerification EventType = "verification"
    EventTypeAcceptance   EventType = "acceptance"
    EventTypeApproval     EventType = "approval"
)

type BaseEvent struct {
    ID        string    `json:"id"`
    Type      EventType `json:"type"`
    Timestamp time.Time `json:"timestamp"`
    PrevHash  string    `json:"prev_hash"`
    Hash      string    `json:"hash"`
}

func (e *BaseEvent) CalculateHash() string
func (e *BaseEvent) Validate() error
```

## Test Strategy
- Unit tests for each event type
- Test hash calculation
- Test validation rules
- Test JSON roundtrip
